# NSW Road Safety Dashboard

Streamlit dashboard for the NSW crash dataset filtered to 2024 onward.

## Files

- `app.py`: Streamlit interface and charts
- `dashboard_data.py`: data loading, filtering, and derived metrics
- `requirements.txt`: Python dependencies

## Run

Install dependencies with:

```powershell
python -m pip install -r requirements.txt
```

Start the app with:

```powershell
python -m streamlit run app.py
```
